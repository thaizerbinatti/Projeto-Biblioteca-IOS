SOBRE NÓS

Alunos de Desenvolvimento Web do Instituto de Oportunidade Social. Foi desenvolvido uma biblioteca utilizando HTML e CSS, que disponibiliza livros de domínio público para leitura online. Vem garantir acesso ao público em geral que admira uma boa leitura, facilitando o acesso a obras literárias mais famosas, de forma gratuita, onde cada usuário pode fazer o download do seu livro em formato PDF, para ler no seu smartphone, tablet ou computador. O projeto permite que o usuário visualize a lista de livros disponíveis na biblioteca, e a capa de cada livro.

  • Tecnologias utilizadas: HTML/CSS.
  • Como executar o projeto: Clone o repositório para sua máquina local. Abra o arquivo index.html em seu navegador de preferência
  • Contribuições: Este é um projeto de código aberto e as contribuições são bem-vindas! Se você deseja contribuir, sinta-se à vontade para fazer um fork e enviar pull requests com melhorias, correções ou novas funcionalidades.

OBJETIVOS

A Biblioteca Digital traz benefícios para todos, possibilitando o acesso à leitura aos usuários. Entre os principais benefícios estão: acesso à cinquenta grandes literaturas brasileiras, estrangeiras, infantil e históricas; facilidade de fazer o download dos livros; com o formato em PDF possibilita a impressão do material; e grande variedade de títulos de livros para escolha.
