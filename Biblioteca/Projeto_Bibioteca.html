    • Nosso Projeto

    Este é um projeto de uma biblioteca online desenvolvido utilizando HTML e CSS, que disponibiliza livros de domínio público para leitura online. Vem garantir acesso ao público em geral que admira uma boa leitura, facilitando o acesso aos mais diversos gêneros literários, de forma gratuita, onde cada usuário pode fazer o download do seu livro favorito para ler no seu smartphone, tablet ou computador. O projeto permite que o usuário visualize a lista de livros disponíveis na biblioteca.

  • Tecnologias utilizadas: HTML/CSS.
  • Como executar o projeto: Clone o repositório para sua máquina local. Abra o arquivo index.html em seu navegador de preferência
  • Contribuições: Este é um projeto de código aberto e as contribuições são bem-vindas! Se você deseja contribuir, sinta-se à vontade para fazer um fork e enviar pull requests com melhorias, correções ou novas funcionalidades.

    • Objetivo da Biblioteca

    A Biblioteca Digital traz benefícios para todos, possibilitando o acesso à leitura aos usuários. Entre os principais benefícios estão: prover o acesso às grandes literaturas brasileiras quanto extrangeira a qualquer hora e em qualquer lugar, possibilitando, o uso simultâneo, por várias pessoas do mesmo objeto digital; recuperação rápida e eficiente da informação; registro da memória da produção institucional; preservação do material em formato digital e impresso, uma vez que o desgaste decorrente do manuseio e das condições climáticas deixará de existir.
